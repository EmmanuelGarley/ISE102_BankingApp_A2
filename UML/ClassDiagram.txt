Relationships:
Bank "1" ---- "many" UserAccount
Program uses Bank
Program uses ConsoleHelper
Bank uses UserAccount

Explanation:
The Bank class manages a collection of UserAccount objects using List<UserAccount>. This supports multiple registered users. The Program class controls the menu flow and delegates business logic to the Bank class. ConsoleHelper supports reusable input handling and reduces repeated code.
